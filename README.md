# Wyo-Hazard | FRC Team 1303 🤖

Official website for **FRC Team 1303 (Wyo-Hazard)**, based in Casper, Wyoming. 

## 🌐 Live Site
Check out our live website here: [wyohazard.com](https://wyohazard.com)

## ⚡ About The Project
This website is a single-page technical portal designed with a "Hazard" industrial aesthetic (Yellow & Black). It showcases our team mission, our seasonal robots, and our community sponsors in Casper.

### Technologies Used
*   **HTML5**: Semantic structure.
*   **CSS3**: Custom "Hazard" styling, CSS Grid, and responsive media queries.
*   **JavaScript**: Smooth scrolling, mobile menu logic, and form handling.
*   **FontAwesome**: For the Biohazard and technical icons.

## 🛠️ Development & Maintenance
To update the site:
1. Edit the `index.html` file.
2. For robot updates, scroll to the `<section id="robots">` and update the card text.
3. For new sponsors, update the `<section id="sponsors">` grid.

## 🏆 Team History
Founded in 2004, Team 1303 has been inspiring students in Casper, Wyoming for over two decades.
